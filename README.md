# Enterprise-RAG-system-Stripe-Docs
Production-Ready RAG System for Stripe Knowledge Bases

Problem Statement:
End users often need fast, accurate answers to Stripe Related questions(like payments, billing, webhooks, rates) from publicly available documentation.
Today, answers are found using manual search and reading pages end to end. This is slow and error prone.
Not to mention there is always a chance of inconsistent guidance and increased time to resolution.

So the aim is to build an enterprise grade Retrieval Augmented Generation system that crawls and cleans Stripe documentation.
Then Semantically indexes text/tables and provides grounded answers.

Success is measured against following criteria :
- >=30% reduction average lookup time
- >= 20% improvement in first answer (human judged)
- >= 85% citations validity rate rate 

All this while keeping monthly inference cost at minimum ($ price to be updated) and complying with security controls.
Next phase will include private data sources (if available/given) and multi-lingual support.
